# Shutter Pilot Brand Icon

Dieses Icon soll beim [Home Assistant Brands Repository](https://github.com/home-assistant/brands) eingereicht werden, damit Shutter Pilot mit Logo in Home Assistant erscheint.

## So fügst du das Icon hinzu

1. Forke https://github.com/home-assistant/brands
2. Kopiere den gesamten Inhalt von `custom_integrations/shutter_pilot/` in deinen Fork:
   - Ziel: `custom_integrations/shutter_pilot/icon.png`
3. Erstelle einen Pull Request

Nach der Merge wird das Icon automatisch in Home Assistant angezeigt.
