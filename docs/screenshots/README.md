# Screenshots für die README

Lege deine drei PNG-Dateien **genau mit diesen Namen** in diesen Ordner:

| Datei | Inhalt |
|-------|--------|
| `dashboard.png` | Tab **Dashboard** (Hauptansicht) |
| `areas.png` | Tab **Bereiche** / **Areas** |
| `shutters.png` | Tab **Rollläden** / **Shutters** |

Die `README.md` und `README.de.md` im Projektroot verlinken diese Dateien.  
**Tipp:** Auflösung z. B. 1200–1600 px Breite, PNG mit sinnvoller Kompression – in der README werden sie klein angezeigt, per Klick öffnet GitHub die volle Größe.

---

**English:** Place your three screenshots here as `dashboard.png`, `areas.png`, and `shutters.png` so the root READMEs display them correctly.
